UTD's Event Nugget Detection and Coreference System at KBP 2015

Jing Lu and Vincent Ng.
Proceedings of the 2015 Text Analysis Conference, 2015.

Click here for the PDF version.

Abstract

We describe UTD's system participated in the event nugget detection and coreference task at TAC-KBP 2015. We designed and implemented a pipeline system that consists of three components: event nugget identification and subtyping, REALIS value identification, and event coreference resolution. In particular, we proposed a multi-pass sieve approach to event coreference resolution. UTD's system achieved F1 scores of 57.45, 45.21, and 32.36 on those three components, respectively.
